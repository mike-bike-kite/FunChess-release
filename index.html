The Game:

Play chess against your computer. It took me quite a time to write this program. I wanted a program that would play like a human and I think I achieved that. You can set up positions, change the appearance of the game etc etc. This game was developed from my 48 hour chess project. Fun chess is totally free but you're welcome to buy me a coffee if you want to say thanks.

To install:

Download the zip file, unzip it and then double click the program to play. If you don't have Java you can download it here.
